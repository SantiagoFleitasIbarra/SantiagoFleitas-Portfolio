# **App Name**: Cosmic Portfolio

## Core Features:

- Dynamic Job Titles: Rotating job titles on the landing page to showcase multiple specializations. Implemented with a smooth animation effect.
- Clear CTAs: Clear 'View Projects' and 'Contact' buttons on the landing page for easy navigation.
- Interactive Timeline: Interactive timeline displaying educational history, with details for each entry.
- Curriculum Download: Downloadable curriculum button in the 'About Me' section.
- Profile Cards: Profile cards on the 'About Me' section.
- Multi-Language Support: Multi-language support with an easy-to-use toggle button. Defaults to Español.
- Credential Summary Tool: Generative AI powered tool: A system for providing summarized AI output for any certification or reference provided by the user for the timeline (including the UdelaR degree), to increase user trust in your background.

## Style Guidelines:

- Primary color: Deep space purple (#4A306D) to evoke the cosmic theme.
- Background color: Near-black (#0A070D) to simulate the darkness of space, providing contrast to UI elements.
- Accent color: Electric indigo (#6F63E8) to highlight interactive elements.
- Headline font: 'Space Grotesk', a sans-serif font with a modern, computerized feel, for titles.
- Body font: 'Inter', a grotesque sans-serif with a clean, neutral look, for the bulk of the text.
- Use icons inspired by cosmic elements (stars, planets, orbits).
- A dynamic layout with parallax scrolling effects to immerse the user in the cosmic theme.
- Subtle animations, such as particles drifting in the background, to enhance the space-like feeling.
